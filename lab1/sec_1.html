RTL设计

RTL设计

下图展示了本节将实现的 SoC.

为了实现上图所示的 SoC, 在已有的代码基础上, 总共需要修改两个部分的硬件设计:

  • 修改总线扩展模块中的译码模块, 使其能对 BRAM 对应的地址进行正确的译码.

  • 在顶层文件中将 RAMCODE 总线接口与总线扩展模块连接.

什么? 你想自己造轮子?

本实验只会教你基于现有的代码基础改进实现 SoC 的设计, 如果你想从完全的裸核开始设计, 可以参考总线扩展模块的设计 (在实验指导书第三章中有详细的介绍) , 实现完总线扩展模块后,向SoC中添加存储指令的 RAM (RAMCODE) , 之后再添加其他外设即可.

本书提供的代码工程名为"CortexM0_SoC", 从本实验开始, 按顺序将实验标记为Task1-4.

首先, 在 "CortexM0_SoC/Task1/rtl/AHBlite_Decoder.v" 文件中修改总线译码模块代码.

  • 在端口参数部分, 令RAMCODE使能有效.

    /*RAMCODE enable parameter*/
parameter Port0_en = 0,
/************************/

    改为：

    /*RAMCODE enable parameter*/
parameter Port0_en = 1,
/************************/

  • 根据第二章所述的 memory map, Cortex-M0 手册推荐 ROM 起始地址分配为 0x00000000, 所以 RAMCODE 的地址编码为0x00000000-0x0000ffff, 所以对于一次总线操作, 只要地址总线的高16位全为0 (总线操作的地址为0x0000xxxx, x在此处表示无论该位为0-f的任何值, 只要高16位全为0, 都不影响总线的译码结果), 则译码模块认为这是一次对指令存储器的操作, 进而生成指令存储器总线选择信号. 因此, 地址译码逻辑只需要对地址的高16位进行判断即可. 所以我们需要在译码模块插入 RAMCODE 有关的译码逻辑. 

    //0x00000000-0x0000ffff
/*Insert RAMCODE decoder code there*/
assign P0_HSEL = 1’b0;
/***********************************/

    改为:

    //0x00000000-0x0000ffff
/*Insert RAMCODE decoder code there*/
assign P0_HSEL = (HADDR[31:16] == 16'h0000) ? Port0_en : 1'b0; 
/***********************************/

    修改完总线译码模块后, 需要在顶层文件(CortexM0SoC.v) 中将RAMCODE总线接口与总线扩展模块连接, 将指令存储器连入SoC中。 在“CortexM0_SoC/Task1/rtl/ CortexM0_SoC.v”中, 已经完成了处理器核、总线扩展模块、RAMCODE总线接口模块以及Block RAM模块的例化(调用模块, 将已有模块添加至设计中) , 但未在总线扩展模块接口部分连接RAMCODE总线接口。因此, 我们需要对代码进行一些简单的修改。

    /* Connect to Interconnect Port 0 */
.HCLK            (clk),
.HRESETn        (cpuresetn),
.HSEL            (/*Port 0*/),
.HADDR            (/*Port 0*/),
.HPROT            (/*Port 0*/),
.HSIZE            (/*Port 0*/),
.HTRANS            (/*Port 0*/),
.HWDATA            (/*Port 0*/),
.HWRITE            (/*Port 0*/),
.HRDATA            (/*Port 0*/),
.HREADY            (/*Port 0*/),
.HREADYOUT        (/*Port 0*/),
.HRESP            (/*Port 0*/),
.BRAM_WRADDR    (RAMCODE_WADDR),
.BRAM_RDADDR    (RAMCODE_RADDR),
.BRAM_RDATA        (RAMCODE_RDATA),
.BRAM_WDATA        (RAMCODE_WDATA),
.BRAM_WRITE        (RAMCODE_WRITE)
/**********************************/

    改为:

    /* Connect to Interconnect Port 0 */
.HCLK            (clk),
.HRESETn        (cpuresetn),
.HSEL            (HSEL_P0),
.HADDR            (HADDR_P0),
.HPROT            (HPROT_P0),
.HSIZE            (HSIZE_P0),
.HTRANS            (HTRANS_P0),
.HWDATA            (HWDATA_P0),
.HWRITE            (HWRITE_P0),
.HRDATA            (HRDATA_P0),
.HREADY            (HREADY_P0),
.HREADYOUT        (HREADYOUT_P0),
.HRESP            (HRESP_P0),
.BRAM_WRADDR    (RAMCODE_WADDR),
.BRAM_RDADDR    (RAMCODE_RADDR),
.BRAM_RDATA        (RAMCODE_RDATA),
.BRAM_WDATA        (RAMCODE_WDATA),
.BRAM_WRITE        (RAMCODE_WRITE)
/**********************************/
