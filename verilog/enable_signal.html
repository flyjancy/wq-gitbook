使能信号

使能信号

什么是使能信号?

使能信号(Enable Signal), 顾名思义, 就是控制某个电路/芯片开启的信号. 我们以 Verilog 中一个常见的多路复用器为例, 

always @(*)
    if (!rstn)begin
        a = 0;
        b = 0;
        c = 0;
        d = 0;
     end
     else 
        case(sel)
        2'b00: a = data_in;
        2'b01: b = data_in;
        2'b10: c = data_in;
        2'b11: d = data_in;
        default :a = 0;
        endcase

在这个模块中, rstn 为我们的使能信号. 当它为 0 的时候, 模块内部的 a, b, c, d 均为 0. 也就是说, 只要 rstn 为 0 时, 不管输入怎么变化 模块的输出均为 0. 此时, 整个模块不工作.

为什么使能信号可以控制模块?

还是以上个模块为例, 我们在 vivado 上综合这个模块, 得到 RTL 电路图. 如下图为例:

电路图

可以看到, rstn 连接一个缓冲器和一个查找表(在这里实现反相器的功能)后, 又连接到各个寄存器的清零端. 所以如果不满足 rstn = 1 的条件的话, 无论输入是什么, 输出均不会变化. 整个模块的选择功能失效.

什么是 LUT? 为什么这里的 LUT 可以实现反相器?

LUT 可以简单的理解为一个存储单元, 具体请见FPGA 简介中FPGA基本结构部分.

LUT 实现反相器的原理为: 利用输入作为地址, 当地址为 0 时, 存储的数据为 1, 当地址为 1 时, 存储的数据为 0.

为什么需要使能信号?

需要使能信号的主要原因是在一个大型的数字系统中, 通常需要多个模块配合完成一个功能. 但是每个模块并不是每时每刻都在工作, 我们需要使用使能信号来控制哪些模块应该工作.

我们仍然以上面提到的选择器为例. 假如我们现在需要一个 3 位的选择器, 如果利用我们已经编写好的模块的话, 则需要使用两个模块实现.

  • 首先, 创建顶层, 并例化mux模块:

     mux u1(
    .sel (sel[1:0]),
    .data_in (data_in),
    .rstn (sel[2]),
    .a (a),
    .b (b),
    .c (c),
    .d (d)
    );
    wire sel2_n;
    assign sel2_n = ~sel[2];

    mux u2(
    .sel (sel[1:0]),
    .data_in (data_in),
    .rstn (sel2_n),
    .a (e),
    .b (f),
    .c (g),
    .d (h)
    );

  • 综合顶层文件之后, 我们可以得到例化之后的电路图:

电路图

可以看到, sel2 连接到了两个子模块的使能端, 这样我们实际上将 3bit 的数据分成了两段, 其中一段从 000 到 011, 另外一段从 100 到 111. 此时如果 sel0 sel1 输入为 11 时, 假设 sel2 输入为 0, 此时 u1 模块启动, u2 模块未被启动, 所以此时 u1 模块的最高位输出. 这样相当于输入为 011时, d 被选择输出; 假设 sel2 输入为 1 时, u1 模块关闭, u2 模块启动, u2模块最高位输出. 相当于输入为 111时, h 被选择输出.

使能信号还有什么作用?

使能信号的用途非常广泛, 可以作为计算机系统中的存储芯片选通信号, 可以作为显示系统中的控制信号. 但核心的一点就是使能信号控制了芯片的功能是否能被正常的启用. 在本次实验中, 我们将会更多的见到使能信号的作用.
